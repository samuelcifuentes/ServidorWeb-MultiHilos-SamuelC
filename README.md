# ServidorWeb-MultiHilos
La aplicación funciona como una página web básica donde los usuarios pueden ingresar desde el navegador, visualizar archivos HTML y ver imágenes JPG o GIF almacenadas en el servidor. Desde la interfaz principal también pueden subir sus propias imágenes, las cuales se guardan en una carpeta del servidor para luego poder visualizarlas. El servidor permanece activo todo el tiempo, atiende varios usuarios al mismo tiempo y muestra en consola las solicitudes que recibe. Si un usuario intenta acceder a un archivo que no existe, el sistema muestra una página de error indicando que el recurso no fue encontrado.
